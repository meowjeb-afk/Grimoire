# Grimoire 📖✨

Grimoire is a modular Windows automation hub and shell customization utility powered by Python. 
Designed for creators who want to reshape their desktop environment, Grimoire runs quietly 
in the system tray, allowing you to cast global hotkeys, automate file pipelines, 
and manifest beautiful transparent desktop widgets.

## 🔮 Active Incantations
* **File Alchemy:** Automatically sorts messy download directories and bulk-renames asset packs sequentially.
* **Clipboard Transmutation:** Instantly intercepts and formats clipboard data via global keyboard hooks.
* **System Scrying:** Layers borderless, custom-themed performance monitors directly onto your desktop wallpaper.
